This folder contains screenshots for the main README documentation.

Place your screenshots here with the following names:

### Main Pages

### Feed & Posts

### User Profiles

## Image Guidelines

- **Format**: PNG preferred (JPEG also acceptable)
- **Size**: Recommended width 1200-1600px for clarity
- **Quality**: High quality, clear text and UI elements
- **Content**: Show key features and functionality
- **Privacy**: Remove or blur any personal/sensitive data

## Taking Screenshots

1. Navigate to the feature/page you want to capture
2. Use Windows Snipping Tool (Win + Shift + S) or similar
3. Save with the exact filename listed above
4. Place in this directory

The README will automatically display your screenshots once you add them here!
